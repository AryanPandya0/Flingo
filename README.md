**Flingo** is a premium, modern, and highly interactive social media platform. Built with a focus on delivering a top-tier user experience, Flingo features a responsive three-column dashboard interface, smooth micro-animations, and full light/dark mode support. 

Whether you're engaging with the community feed or exploring user profiles, Flingo is designed to look stunning and perform flawlessly across all devices.

## 🌟 Key Features

- **Modern Three-Column UI:** A premium desktop layout featuring a sidebar navigation, a central interactive content feed, and a right-side activity/suggestions panel.
- **Dynamic Theming:** Built-in seamless switching between Light and Dark modes.
- **Micro-Animations:** Powered by Framer Motion, providing buttery-smooth transitions, hover effects, and interactive feedback.
- **Fully Responsive:** Carefully crafted to adapt from ultra-wide monitors down to mobile screens without sacrificing aesthetic quality.
- **High-Performance:** Leveraging Vite and React 19 for lightning-fast hot module replacement (HMR) and optimized production builds.
- **Modern Iconography:** Clean and consistent iconography provided by Lucide React.

## 🛠️ Technology Stack

- **Frontend Framework:** React (v19)
- **Build Tool:** Vite (v8)
- **Routing:** React Router DOM (v7)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Utilities:** clsx, tailwind-merge (for dynamic class name composition)

## 🚀 Getting Started

Follow these steps to run the Flingo development environment locally:

### 1. Clone the repository
```bash
git clone https://github.com/AryanPandya0/Flingo.git
cd Flingo/frontend
```

### 2. Install dependencies
```bash
npm install
```

### 3. Start the development server
```bash
npm run dev
```

### 4. Open in browser
Open your browser and navigate to `http://localhost:5173` to see Flingo in action!
